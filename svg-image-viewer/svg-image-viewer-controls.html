About the Controls

The SVG Image Viewer component now supports an optional controls attribute that adds a familiar control interface similar to HTML5 media elements.

Keyboard Shortcuts (when component has focus):

  • Arrow keys: Pan the image when pan mode is enabled
  • + / -: Zoom in and out
  • Home / 0: Reset to fit view
  • Space: Toggle pan mode
  • f: Toggle fullscreen mode

Sample Code:

<svg-image-viewer src="sample.jpg" controls></svg-image-viewer>
